# Veri Görselleştirme ve Harita Projesi

Bu projede Python kullanılarak veri analizi, görselleştirme ve harita oluşturma işlemleri gerçekleştirilmiştir.

## Kullanılan Teknolojiler

* Python
* Pandas
* Matplotlib
* Seaborn
* Folium

## Proje İçeriği

* Öğrenci performans verileri analiz edilmiştir
* Çizgi, dağılım, çubuk ve pasta grafikleri oluşturulmuştur
* Korelasyon analizi ile değişkenler arasındaki ilişkiler incelenmiştir
* Folium kütüphanesi kullanılarak Türkiye haritası üzerinde şehir bazlı görselleştirme yapılmıştır
* Şehirler arası bağlantılar çizilmiş ve belirli bölgeler vurgulanmıştır

## Çıktılar

* Veri görselleştirme grafiklerinin oluşturulması
* Etkileşimli Türkiye haritası (turkiye_harita.html)
* Analiz sonuçlarının yorumlanması

## Projeyi Çalıştırma

1. Gerekli kütüphaneleri yükleyin:
   pip install pandas matplotlib seaborn folium

2. Python dosyasını çalıştırın:
   python main.py

3. Harita çıktısı "turkiye_harita.html" dosyası olarak oluşacaktır.
